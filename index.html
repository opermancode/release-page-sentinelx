Detect. Get. Operate.

A high-speed Go orchestrator that finds and manages your fleet with zero friction. NeuRader is a single-binary power tool for modern sysadmins.

$ curl -L https://neurader.operman.in/releases/neurader-latest -o neurader && chmod +x neurader && ./neurader --help

How it Works.

01 / Setup

Deploy the Jumpbox. Generate your Master SSH keys automatically. One binary, zero config files to start.

02 / Handshake

Nodes signal the Jumpbox via our encrypted registration server or manual key drop. No complex agents required.

03 / Blast

Run commands across 100+ nodes in parallel. Stream logs back to your terminal in real-time with zero lag.

Intelligent Handshake

Automated SSH key orchestration. Connect new nodes to your Jumpbox in under 5 seconds with built-in security validation.

Static Performance

Built with Go. No Python dependencies or heavy runtimes. A 15MB binary that works on any Linux distribution instantly.

Real-time Radar

Simultaneous command execution with live-streaming output. Watch your entire infrastructure respond to your commands in sync.

Cascading Upgrades.

Tired of manual updates? NeuRader pulls the latest build from the dev server and blasts the update to every child node in your inventory automatically.

$ neurader upgrade
