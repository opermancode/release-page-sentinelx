Documentation

Sentinex is a lightweight, agent-based orchestrator designed for secure, parallel command execution across Linux fleets. It uses Go's native concurrency to outperform traditional Python-based tools.

Installation

Run the universal installer on any supported Linux distribution (Ubuntu, Debian, CentOS, RHEL).

curl -sSL sentinex.io/get | bash
Note: The installer requires sudo privileges to move the binary to /usr/local/bin and register the systemd service.

The Jumpbox Role

The Jumpbox (Manager) is the command center. Once installed, run the setup wizard and select Role 1.

sudo sentinex install

The Jumpbox will generate a master SSH key pair and start a background listener on port 9090 to receive registration requests from Child nodes.

The Child Role

Child nodes are the target servers. Run the installer and select Role 2. You will be asked for the IP of your Jumpbox to begin the secure handshake.
